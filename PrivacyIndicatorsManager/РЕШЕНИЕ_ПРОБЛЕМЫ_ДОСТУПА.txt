========================================
РЕШЕНИЕ ПРОБЛЕМЫ ДОСТУПА К МИКРОФОНУ
========================================

ПРОБЛЕМА:
---------
После отключения индикаторов приватности появилась ошибка:
"Отказано в доступе к микрофону (0x80070005)"

ПРИЧИНА:
--------
Отключение индикаторов приватности через реестр может также
заблокировать доступ к микрофону для приложений.

РЕШЕНИЕ:
--------

ВАРИАНТ 1 (БЫСТРЫЙ):
---------------------
1. Найдите файл: ENABLE_MICROPHONE_ACCESS.cmd
2. Правой кнопкой мыши -> "Запуск от имени администратора"
3. Следуйте инструкциям
4. Перезапустите приложение AudioRecorder

ВАРИАНТ 2 (ВРУЧНУЮ):
---------------------
1. Откройте Параметры Windows (Win+I)
2. Конфиденциальность и защита -> Микрофон
3. Включите "Разрешить приложениям доступ к микрофону"
4. Включите "Разрешить классическим приложениям доступ к микрофону"
5. Перезапустите приложение

ВАРИАНТ 3 (POWERSHELL):
------------------------
1. Запустите PowerShell от имени администратора
2. Выполните: .\FIX_MICROPHONE_ACCESS.ps1
3. Перезапустите приложение

ЧТО ДЕЛАЮТ СКРИПТЫ:
------------------
✓ Включают доступ к микрофону через CapabilityAccessManager
✓ Настраивают доступ для классических приложений
✓ Включают доступ через групповые политики (если доступно)

ВАЖНО:
------
- После выполнения скрипта ПЕРЕЗАПУСТИТЕ приложение
- Если не помогает - перезагрузите компьютер
- Убедитесь, что микрофон включен в настройках Windows
