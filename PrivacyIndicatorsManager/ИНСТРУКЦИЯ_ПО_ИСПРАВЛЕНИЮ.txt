========================================
ИНСТРУКЦИЯ ПО ОТКЛЮЧЕНИЮ ИНДИКАТОРА МИКРОФОНА
========================================

ПРОБЛЕМА:
---------
Скрипты устанавливают значение в реестре, но индикатор все равно показывается.

РЕШЕНИЕ:
--------

ВАРИАНТ 1 (РЕКОМЕНДУЕТСЯ):
---------------------------
1. Откройте папку: PrivacyIndicatorsManager
2. Найдите файл: CHECK_AND_FIX.cmd
3. Правой кнопкой мыши -> "Запуск от имени администратора"
4. Скрипт проверит текущее состояние и предложит исправления
5. Следуйте инструкциям на экране
6. ОБЯЗАТЕЛЬНО перезагрузите компьютер после выполнения!

ВАРИАНТ 2 (ПОЛНОЕ ИСПРАВЛЕНИЕ):
--------------------------------
1. Найдите файл: DISABLE_MIC_INDICATOR_FIXED.cmd
2. Правой кнопкой мыши -> "Запуск от имени администратора"
3. Этот скрипт использует несколько методов одновременно
4. ОБЯЗАТЕЛЬНО перезагрузите компьютер после выполнения!

ВАЖНО:
------
✓ Запускайте скрипты ОТ ИМЕНИ АДМИНИСТРАТОРА
✓ ПЕРЕЗАГРУЗИТЕ компьютер после выполнения (не только Explorer!)
✓ На Windows Home может работать не полностью
✓ Некоторые обновления Windows могут сбрасывать настройки

ПОЧЕМУ НЕ РАБОТАЕТ:
------------------
1. Значение в реестре установлено, но система не перезагружена
2. Windows 11 требует дополнительных параметров
3. На Windows Home групповые политики могут не работать
4. Служба конфиденциальности не перезапущена

ЧТО ДЕЛАЮТ НОВЫЕ СКРИПТЫ:
-------------------------
✓ Проверяют текущее состояние реестра
✓ Устанавливают все необходимые параметры
✓ Применяют дополнительные настройки для Windows 11
✓ Перезапускают необходимые службы
✓ Предлагают перезагрузить компьютер

ЕСЛИ ВСЕ РАВНО НЕ РАБОТАЕТ:
---------------------------
1. Убедитесь, что перезагрузили компьютер (не только Explorer!)
2. Проверьте версию Windows (нужна 1903+)
3. На Windows Home используйте виртуальный аудио драйвер
4. Некоторые версии Windows 11 могут не поддерживать отключение
