========================================
Gizlilik gostergelerinin idare edilmesi
========================================

Bu paket Windows-da gizlilik gostergelerini idare etmek ucun skriptler ehtiva edir.

Fayllar:
--------
1. manage_privacy_indicators.cmd - Esas idare skripti (ISTIFADE EDIN)
2. check_privacy_indicators_support.cmd - Veziyyetin yoxlanilmasi
3. enable_privacy_indicators.cmd - Gostergelerin aktivlesdirilmesi
4. disable_privacy_indicators.cmd - Gostergelerin sonderilmesi
5. check_windows_version.cmd - Windows versiyasinin yoxlanilmasi

Istifade:
---------
1. manage_privacy_indicators.cmd faylini sag duymeni basib "Administrator kimi ise sal" secerek ise salin.

2. Menyudan secim edin:
   - 1: Veziyyeti yoxla - sistemin desteyini ve movcud veziyyeti gosterir
   - 2: Gostergeleri sonder - gizlilik gostergelerini sonderir
   - 3: Gostergeleri aktivlesdir - gizlilik gostergelerini aktivlesdirir
   - 4: Cixis - skripti baglayir

Desteklenen sistemler:
----------------------
- Windows 10 versiya 1903 (qurulum 18362) ve daha yenileri
- Windows 11 (butun versiyalar)

Terebler:
---------
- Administrator huquqlari
- Windows 10/11 Pro, Enterprise, Education (tam destek)
- Windows 10/11 Home (isleye biler, lakin garantiya verilmir)

Qeydler:
--------
- Deyisikliklerin tetbiqi ucun Explorer-i yeniden baslatmaq ve ya kompyuteri yeniden yuklemek lazımdır.
- Bu reestr acari Microsoft terefinden resmi olaraq senedlesdirilmeyib.
- Windows Home versiyalarinda qrup siyaseti desteyi olmadığı ucun tesir gostermeye biler.
