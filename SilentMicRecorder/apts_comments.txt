SrvC - main configuration section of the AudioCore service

DS  - Duration Seconds: length of one recording segment in seconds
OF  - Output Folder: folder where audio recordings are saved
SR  - Sample Rate: sampling rate in Hz (e.g. 44100)
BPS - Bits Per Sample: sample bit depth (usually 16)
Ch  - Channels: number of channels (1 = mono, 2 = stereo)
AF  - Audio Format: audio format code (0 = wav, 1 = mp3)
