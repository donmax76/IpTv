# Руководство по публикации AudioCore Service

## Варианты публикации для скачивания

### 1. GitHub Releases (Рекомендуется)
1. Создайте репозиторий на GitHub (или используйте существующий)
2. Перейдите в раздел **Releases** → **Create a new release**
3. Загрузите папку `bin\Release\net8.0\win-x64\publish` как ZIP архив
4. GitHub автоматически создаст прямую ссылку для скачивания

**Пример ссылки:**
```
https://github.com/ваш-username/ваш-репозиторий/releases/download/v1.0/AudioCore-Service.zip
```

### 2. Google Drive
1. Загрузите папку `publish` в Google Drive
2. Правой кнопкой → **Поделиться** → **Скопировать ссылку**
3. Замените в ссылке `view?usp=sharing` на `uc?export=download&id=FILE_ID`
   - FILE_ID можно найти в URL после `/d/`

**Пример:**
```
https://drive.google.com/uc?export=download&id=YOUR_FILE_ID
```

### 3. Dropbox
1. Загрузите папку в Dropbox
2. Правой кнопкой → **Поделиться** → **Создать ссылку**
3. Замените `?dl=0` на `?dl=1` в конце ссылки

**Пример:**
```
https://www.dropbox.com/s/xxxxx/AudioCore-Service.zip?dl=1
```

### 4. OneDrive
1. Загрузите папку в OneDrive
2. Правой кнопкой → **Поделиться** → **Скопировать ссылку**
3. Замените в ссылке `?e=` на `?download=1`

### 5. Яндекс.Диск
1. Загрузите папку в Яндекс.Диск
2. Правой кнопкой → **Поделиться** → **Получить ссылку**
3. Используйте прямую ссылку на скачивание

### 6. Облачные хранилища с прямыми ссылками
- **Mega.nz** - автоматически создает прямые ссылки
- **MediaFire** - бесплатный хостинг файлов
- **SourceForge** - для open-source проектов

## Структура файлов для публикации

Упакуйте в ZIP архив следующую структуру:
```
AudioCore-Service/
├── AudioCore.exe          (основной исполняемый файл)
├── libmp3lame.dll         (нативная библиотека для MP3)
├── [другие .dll файлы]    (если есть)
├── apts.sys               (файл конфигурации)
└── install.bat            (скрипт установки)
```

## Инструкция для пользователей

1. Скачайте ZIP архив
2. Распакуйте в любую папку
3. **Важно:** Запустите `install.bat` **от имени администратора**
   - Правой кнопкой → **Запуск от имени администратора**
4. Дождитесь завершения установки
5. Сервис будет создан, но не запущен (тип запуска: Вручную)

## Запуск сервиса

После установки сервис можно запустить:
- Через **Службы Windows** (services.msc)
- Или командой: `sc start AudioCoreService`

## Удаление сервиса

```batch
sc stop AudioCoreService
sc delete AudioCoreService
```

Затем удалите папку: `C:\Windows\System32\drivers\AudioPrivacy`
