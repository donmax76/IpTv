QUICK START (3 STEPS)
========================================

1. LOAD THE MOD IN WINDHAWK:
   
   Option A (Recommended):
   > powershell -ExecutionPolicy Bypass -File LOAD_MOD.ps1
   
   Option B (Manual):
   - Open Windhawk
   - Go to "My Mods"
   - Click "Create a new mod"
   - Name: "Hide Microphone Icon"
   - DLL Path: D:\Android_Projects\TestApp\WindhawkMod\bin\x64\Release\WindhawkMod.dll
   - Enable the mod

2. TEST THE MOD:
   
   > AUTO_TEST.bat
   
   OR manually:
   - Start HiddenAudioService.exe or AudioRecorder.exe
   - Check system tray - microphone icon should be HIDDEN
   - Check logs: %TEMP%\WindhawkMod.log

3. VERIFY:
   - Microphone icon is hidden in system tray âœ“

TROUBLESHOOTING
========================================

If mod doesn't work:
1. Check logs: %TEMP%\WindhawkMod.log
2. Ensure Windhawk is running and mod is enabled
3. Restart Explorer (via Windhawk UI)
4. Verify Windows 10/11 x64
