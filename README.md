# ТВ Онлайн — Android приложение для просмотра IPTV

Приложение для просмотра телеканалов онлайн на Android с поддержкой M3U плейлистов.

## Возможности

- Просмотр IPTV каналов в реальном времени
- Встроенные плейлисты (IPTV-ORG, Free-IPTV)
- Поддержка стран: Россия, Украина, Беларусь
- Категории: спорт, новости, музыка
- Горизонтальная панель каналов
- ExoPlayer для стабильного воспроизведения

## Встроенные плейлисты

Приложение включает рабочие плейлисты с открытых источников:

| Плейлист | Описание |
|----------|----------|
| IPTV-ORG (Общие) | Общий список каналов |
| IPTV-ORG (Россия) | Российские каналы |
| IPTV-ORG (Украина) | Украинские каналы |
| IPTV-ORG (Беларусь) | Белорусские каналы |
| IPTV-ORG (Спорт) | Спортивные каналы |
| IPTV-ORG (Новости) | Новостные каналы |
| IPTV-ORG (Музыка) | Музыкальные каналы |
| Free-IPTV (Русские) | Русские каналы |

Источники: [iptv-org](https://github.com/iptv-org/iptv), [Free-IPTV](https://github.com/Free-IPTV)

## Требования

- Android 7.0 (API 24) или выше
- Подключение к интернету

## Сборка

1. Установите [Android Studio](https://developer.android.com/studio) или Android SDK
2. Создайте файл `local.properties` в корне проекта:
   ```
   sdk.dir=/path/to/your/Android/Sdk
   ```
3. Соберите APK:
   ```bash
   ./gradlew assembleDebug
   ```
4. APK будет в `app/build/outputs/apk/debug/app-debug.apk`

## Установка

Скопируйте `app-debug.apk` на устройство и установите (разрешите установку из неизвестных источников).

## Использование

1. Запустите приложение
2. Выберите плейлист в выпадающем списке
3. Дождитесь загрузки каналов
4. Нажмите на канал для просмотра

## Технологии

- Kotlin
- AndroidX Media3 (ExoPlayer)
- OkHttp
- Coroutines

## Лицензия

Образовательный проект. Плейлисты — из открытых источников.
